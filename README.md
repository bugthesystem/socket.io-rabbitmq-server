# socket.io-rabbitmq-server
socket.io pub-sub server [rabbitmq](https://www.rabbitmq.com/) implementation


# Installing

```
$ npm install socket.io-rabbitmq-server
```


# Usage

```shell
$ socket.io-rabbitmq-server
```


# Testing

```shell
$ make test
```
